Build a complete, modern, responsive web application called "Smart Curriculum Activity & Attendance App".

Technology:
- React + TypeScript
- Tailwind CSS
- Node.js + Express
- SQLite (or PostgreSQL if available)
- Modern dashboard UI
- Mobile and desktop responsive

Project Overview:
The application is designed for colleges to automate attendance, manage students, display the latest Anna University syllabus, and help students use their free periods productively.

Main Features

1. Authentication
- Faculty Login
- Student Login
- Admin Login
- Secure password authentication
- Role-based dashboard

2. Dashboard
- Total Students
- Today's Attendance
- Attendance Percentage
- Subjects
- Notifications
- Recent Activities
- Quick Actions

3. Student Management
- Add Student
- Edit Student
- Delete Student
- Search Student
- Filter by Department, Year and Section
- Student Profile with
  - Photo
  - Name
  - Register Number
  - Department
  - Year
  - Section
  - Email
  - Phone Number

Support at least 40 students per class.

4. Attendance System
Support multiple attendance methods:
- Face Recognition
- QR Code
- Manual Attendance

Features:
- Present
- Absent
- Late
- Date and Time
- Attendance History
- Attendance Percentage
- Monthly Reports

5. Face Recognition
Use browser camera.

Requirements:
- Register each student's face.
- Detect only real human faces.
- Prevent fake photo attendance if possible.
- Automatically mark attendance after successful recognition.
- Show success message.

6. QR Attendance
Generate QR code for each class.
Students scan QR.
Attendance recorded automatically.

7. Timetable
Faculty can create timetable.
Students can view today's timetable.
Highlight current class.

8. Anna University Curriculum

Display
